  • 1、有哪些标签？

    body / div / span / p / a / br / select / h1 / head / img / option / title / label / input / textarea / button / table /...

  • 2、行内元素和块状元素？

    块状元素：是指开始在新的行，占领整行宽度。

    行内元素：仅仅维持内容的宽度。

  • 3、属性？

    总是以key/value对的形式出现。

    声明class、id、name、style、data-*等。

  • 4、HTML5？

    提供更多语义化标签。

    header / nav / article / section / aside / footer / ...

JAVASCRIPT

前世今生

  • 1995年，Netscape（网景）公司发布LiveScript。
  • 同年，改名Javascript。
  • 1997年，Javascript 1.1版本移交给 ECMA 组织。
  • 2009年，ECMA 组织发布 ECMAScript 5.0。
  • 2015年，ECMA 组织发布 ECMAScript 6.0。

React
  1. 用户访问 View
  2. View 发出用户的 Action
  3. Dispatcher 收到 Action，要求 Store 进行相应的更新
  4. Store 更新后，发出一个"change"事件
  5. View 收到"change"事件后，更新页面
